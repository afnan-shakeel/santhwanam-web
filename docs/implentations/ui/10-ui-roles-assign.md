## Summary

| Component | Description |
|-----------|-------------|
| **Main Modal** | Single modal with view/add modes |
| **View Mode** | User info + current roles list + [+ Add] button |
| **Add Mode** | Inline form at top, roles list below |
| **Confirmation Modal** | Only for revoke action |
| **Role Dropdown** | Grouped by system/custom with scope indicator |
| **Scope Dropdown** | Hierarchical, only shown for scoped roles |
