**Version:** 2.0  
**Last Updated:** January 2025  
**Status:** Final

---

## Key Design Decisions

### 1. Auto-Debit for Contributions
Wallet debits are **auto-processed** when a contribution cycle starts. No agent acknowledgment required.

**Configuration:** `wallet.autoDebitEnabled` flag controls this behavior.

### 2. Tabbed Layout for Member Wallet
Member wallet uses a **single page with tabs** instead of separate pages. Balance remains visible across all tabs.

### 3. Shared Components with Role-Specific Wrappers
Two page-level components (MyWallet for members, MemberWallet for agent/admin) share presentational components.

  recentTransactions: WalletTransaction[];

  pendingCashContributions?: PendingContribution[];
}
