# Agent Profile UI Design

---

## **Overview**

Agent profile should show:
1. Personal/professional information
2. Performance metrics
3. Members under management
4. Commission/earnings (if applicable)
5. Activity history
6. Territory/hierarchy info

---

## **Recommended Structure: Tabbed Layout**

### **URL:** `/agents/:agentId/profile` or `/my-profile` (for logged-in agent)

## **Summary**

**5 Tabs:**
1. **Overview** - Quick stats, recent activity
2. **Members** - List of members under agent
3. **Performance** 
4. **Collections** - Outstanding + history [Implement in future]
5. **Activity Log** - Full audit trail [implement in future]

**Key Features:**
- Visual hierarchy stats
- Performance tracking
- Member management
- Collection monitoring (Future implement)
- Activity history (Future implement)

**Mobile-responsive** with simplified tab navigation
