**Version:** 2.0  
**Last Updated:** January 2025  
**Status:** Final

---

## Overview

This document covers the complete Agent Profile UI including:
- Agent viewing their own profile (1st person / self view)
- Admin viewing an agent's profile (3rd person / admin view)
- Shared components and design patterns
- Navigation flows and API endpoints

---

## Key Design Decisions

### 1. Consistent Design Pattern
Agent Profile follows the same design pattern as Member Profile and Member Wallet:
- Persistent header with info card
- Tab navigation for organized content
- ViewMode controls actions and visibility

### 2. Two Page Components, Shared Presentational Components
- `MyAgentProfileComponent` â€” Agent's own view
- `AgentProfileComponent` â€” Admin view (with viewMode)
- Shared components for consistent UI

### 3. Agent Profile as Landing Page
For agents, `/agent/profile` serves as the landing/home page (no separate dashboard for now).

## Route Configuration

```typescript
const routes: Routes = [
  // Agent self routes
  {
    path: 'agent/profile',
    component: MyAgentProfileComponent,
    canActivate: [authGuard, agentGuard],
    children: [
      { path: '', redirectTo: 'overview', pathMatch: 'full' },
      { path: 'overview', component: AgentOverviewTabComponent },
      { path: 'members', component: AgentMembersTabComponent },
      { path: 'collections', component: AgentCollectionsTabComponent },
      { path: 'performance', component: AgentPerformanceTabComponent }
    ]
  },
  
  // Agent member routes
  {
    path: 'agent/members/:memberId',
    component: MemberProfileComponent,
    data: { viewMode: 'agent' },
    canActivate: [authGuard, agentGuard],
    // ... children
  },
  {
    path: 'agent/members/:memberId/wallet',
    component: MemberWalletComponent,
    data: { viewMode: 'agent' },
    canActivate: [authGuard, agentGuard],
    // ... children
  },
  {
    path: 'agent/contributions/:contributionId/collect',
    component: CollectContributionComponent,
    canActivate: [authGuard, agentGuard]
  },
  
  // Admin agent routes
  {
    path: 'agents',
    component: AgentListingComponent,
    canActivate: [authGuard, adminGuard]
  },
  {
    path: 'agents/:agentId',
    component: AgentProfileComponent,
    data: { viewMode: 'admin' },
    canActivate: [authGuard, adminGuard],
    children: [
      { path: '', redirectTo: 'overview', pathMatch: 'full' },
      { path: 'overview', component: AgentOverviewTabComponent },
      { path: 'members', component: AdminAgentMembersTabComponent },
      { path: 'collections', component: AdminAgentCollectionsTabComponent },
      { path: 'performance', component: AdminAgentPerformanceTabComponent },
      { path: 'activity', component: AdminAgentActivityTabComponent }
    ]
  }
];
```
