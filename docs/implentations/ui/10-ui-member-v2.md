# Member Profile - Complete UI Specification

**Version:** 2.0  
**Last Updated:** January 2025  
**Status:** Final

---

## Overview

This document covers the complete Member Profile UI including:
- Member viewing their own profile (1st person / self view)
- Agent viewing a member's profile (3rd person / agent view)
- Admin viewing a member's profile (3rd person / admin view)
- Shared components and design patterns
- Navigation flows and API endpoints

---

## Key Design Decisions

### 1. Consistent Design Pattern
Member Profile follows the same design pattern as Agent Profile and Member Wallet:
- Persistent header with info card
- Tab navigation for organized content
- ViewMode controls actions and visibility

### 2. Three View Modes
- `self` â€” Member viewing their own profile
- `agent` â€” Agent viewing their assigned member
- `admin` â€” Admin with full access

### 3. Wallet as Link, Not Tab
Wallet is accessed via link in Quick Info card, navigating to the dedicated wallet page.

### 4. Document Upload â€” Agent Only
Members cannot upload documents themselves. Agents upload on behalf of members.

## Route Configuration

```typescript
const routes: Routes = [
  // Member self routes
  {
    path: 'my-profile',
    component: MyProfileComponent,
    canActivate: [authGuard, memberGuard],
    children: [
      { path: '', redirectTo: 'overview', pathMatch: 'full' },
      { path: 'overview', component: MemberOverviewTabComponent },
      { path: 'contributions', component: MemberContributionsTabComponent },
      { path: 'nominees', component: MemberNomineesTabComponent },
      { path: 'documents', component: MemberDocumentsTabComponent }
    ]
  },
  
  // Agent viewing member
  {
    path: 'agent/members/:memberId',
    component: MemberProfileComponent,
    data: { viewMode: 'agent' },
    canActivate: [authGuard, agentGuard],
    children: [
      { path: '', redirectTo: 'overview', pathMatch: 'full' },
      { path: 'overview', component: MemberOverviewTabComponent },
      { path: 'contributions', component: AgentMemberContributionsTabComponent },
      { path: 'nominees', component: MemberNomineesTabComponent },
      { path: 'documents', component: AgentMemberDocumentsTabComponent }
    ]
  },
  
  // Admin viewing member
  {
    path: 'admin/members/:memberId',
    component: MemberProfileComponent,
    data: { viewMode: 'admin' },
    canActivate: [authGuard, adminGuard],
    children: [
      { path: '', redirectTo: 'overview', pathMatch: 'full' },
      { path: 'overview', component: MemberOverviewTabComponent },
      { path: 'contributions', component: MemberContributionsTabComponent },
      { path: 'nominees', component: AdminMemberNomineesTabComponent },
      { path: 'documents', component: AdminMemberDocumentsTabComponent },
      { path: 'activity', component: AdminMemberActivityTabComponent }  // Future
    ]
  }
];
```
