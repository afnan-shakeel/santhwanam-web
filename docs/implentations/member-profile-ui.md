# Member Profile UI Design

---

## **Overview**

Member profile should show:
1. Personal & contact information
2. Membership details & status
3. Nominees information
4. Wallet balance
5. Contribution history (Future implementation)
6. Documents 
7. Activity timeline (Future implementation)

---

## **Recommended Structure: Card-Based Layout with Tabs**

### **URL:** `/members/:memberId/profile` or `/my-profile` (for logged-in member)
